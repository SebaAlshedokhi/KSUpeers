## Abstract
This project aims to build an Android app for students in the Diploma of Programming and Databases at King Saud University (KSU). The app lets talented students become peer helpers, sharing their easy-to-understand study summaries and notes for a small fee. This setup helps learners get clearer explanations from fellow students and gives the student-helpers a simple way to earn extra income.
